- [ ] **Step 5: hook 동작 수동 검증 (실제 commit 없이)**

```bash

printf '\n// trailing test   \n' >> src/app/page.tsx
git add src/app/page.tsx

# 검증: prettier가 끝 공백 제거하고 변경분이 다시 stage됐는지 확인
git diff --cached src/app/page.tsx | grep "trailing test" && \
  ! git diff --cached src/app/page.tsx | grep -E "trailing test   $" && \
```
